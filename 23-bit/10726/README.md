# 10726. 이진수 표현

## 문제 출처
[SWEA](https://swexpertacademy.com/main/code/problem/problemDetail.do?contestProbId=AXRSXf_a9qsDFAXS#none)

## 💡 접근 방식

### 1. 사용 알고리즘
* **비트마스킹(Bitmasking)**
* **비트 연산(Bitwise Operation)**

### 2. 문제 풀이 과정
1.  문제의 핵심은 정수 `M`의 이진수 표현에서 마지막 `N`개의 비트가 모두 1로 켜져 있는지 확인하는 것입니다.
2.  이를 확인하기 위해, 마지막 `N`개의 비트가 모두 1로 설정된 **비트마스크** `mask`를 생성합니다.
3.  `mask`는 `1 << N` (1을 `N`번 왼쪽 시프트, 예: `N=3`이면 `1000`)에서 1을 빼서 만듭니다. 즉, `(1 << N) - 1`은 2진수로 `N`개의 1이 연속된 값(예: `N=3`이면 `111`)이 됩니다.
4.  `M`의 마지막 `N` 비트가 모두 1인지 판별하기 위해 **비트 OR 연산자(`|`)**를 활용합니다.
5.  `M | mask` 연산을 수행합니다. 만약 `M`의 마지막 `N` 비트가 이미 모두 1이었다면, 1로만 채워진 `mask`와 OR 연산을 해도 `M`의 값은 변하지 않습니다.
6.  따라서, `(M | mask) == M` 조건이 참(True)이면, `M`의 마지막 `N` 비트가 모두 1이라는 의미이므로 "ON"을 출력합니다.
7.  만약 `M`의 마지막 `N` 비트 중 하나라도 0이 있었다면, `mask`와의 OR 연산 시 해당 비트가 1로 바뀌어 `(M | mask)`의 결과가 `M`과 달라지게 됩니다. 이 경우 "OFF"를 출력합니다.
